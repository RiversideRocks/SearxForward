Searx Redirector Tool

Forwards your search to a random Searx instance.

Adding to Firefox

  1. Right click on the URL bar
  2. Select the add option at the bottom of the menu
  3. Open settings
  4. Scroll to search engines
  5. Select Searx redirector as your default search engine

Adding to Chrome

  1. Right click on the URL bar
  2. Click manage search engines
  3. Click add
  4. Set the name to SearxRedirector
  5. Leave keyword blank
  6. Set url to "https://THISINSTANCEURL/?q=%s"
  7. Leave last box blank
